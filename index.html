Introdução

Vamos falar sobre uma linguagem de marcação. O html (HyperText Markup Language) é a linguagem padrão usada para criar páginas web. É como a estrutura esquelética de um site, fornecendo a base para o conteúdo, que pode incluir:

  • Textos
  • Imagens
  • Vídeos
  • Links

Para outras páginas. Html utiliza "tags" ou "marcadores" que dizem ao navegador da web como exibir cada elemento da página. Por exemplo, a tag "p" é usada para parágrafos, "img" para imagens e "a" para links. Combinado com Css (Cascading Style Sheets) para estilização e JavaScript para interatividade, html permite construir sites atrativos e funcionais.

História

Html começou a ganhar popularidade na década de 1990, especialmente com a introdução do html 2.0 em 1995, que se tornou um padrão para a criação de páginas web. Desde então, html evoluiu e continua sendo a base da maioria dos sites na internet.
